# Sharp Shapes

A Python 3 CLI program to calculate the area occupied by a shape.

Run `python sharpshapes.py`


#### Example output
```
Select a shape:
1. Rectangle
2. Circle
3. Cuboid
4. Cylinder
9. Quit
> cub

Enter the width, height, depth separated by stuff.
> 2 4  5
The total volume of the cuboid is 40.0
```
